# Burger Builder App

This is a Burger Builder application built using React, allowing users to customize and order their own burgers.


## Installation

To run the Tic Tac Toe game locally, follow these steps:

1. Clone the repository:

```
  git clone https://github.com/abbasifaizan70/Burger-Make-App
```

2. Navigate to the project directory:

```
  cd Burger-Make-App
```

3. Install the dependencies:

```
  npm install
```

4. Start the development server:

```
  npm start
```


5. Open [http://localhost:3000](http://localhost:3000) to view the game in your browser.
